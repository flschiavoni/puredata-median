# puredata-median
Median~ is a Pure Data external to calculate the median of a signal 

![My image](https://github.com/flschiavoni/puredata-median/blob/gh-pages/median.png)

TO COMPILE
==========
`make`

TO INSTALL
==========
`make install`

(In Linux you should use `sudo make install`)


TO INSTALL ONLY TO USER (Linux)
========================

`mkdir -p ~/pd-externals/median`

`cp median.pd_linux ~/pd-externals/median/.`

`cp median_help.pd  ~/pd-externals/median/.`

`cp median-meta.pd  ~/pd-externals/median/.`

FILES INFORMATION
=================

Makefile is based on oficial Pure Data Makefile Template available in:

https://puredata.info/docs/developer/MakefileTemplate

Header files (.h) was copied from Pure Data oficial repository available in:

https://svn.code.sf.net/p/pure-data/svn/trunk/pd/src/
